当人们打开系统的网址后，首先看到的就是首页界面。在这里，人们能够看到智慧学生校舍系统的导航条，通过导航条导航进入各功能展示页面进行操作。系统首页界面如图5-1所示：

![](/md/blog.017.png)

图5-2 系统注册界面

系统登录，在登录页面选择权限，在正确输入用户名和密码后，进入操作系统进行操作；如图5-3所示。

图5-3系统登录界面

![](/md/blog.019.png)

图5-4公告物品详情页面

![](/md/blog.020.png)

图5-5失物招领详情页面

个人中心：在个人中心页面可以更新个人详情信息，还可以对我的发布和我的收藏页面进行操作；如图5-6所示：
